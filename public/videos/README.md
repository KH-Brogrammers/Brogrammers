# Project Videos

Place your project video files here.

## Naming Convention

Match the video filenames with the project IDs in `mockData.ts`:

- `tagglabs.mp4`
- `emotion-design.mp4`
- `ai-photo-booth.mp4`
- `sc-exhibit-1.mp4`
- `sc-exhibit-2.mp4`
- `sc-exhibit-3.mp4`
- `sc-exhibit-4.mp4`
- `sc-exhibit-5.mp4`

## Format

- Preferred: MP4 (H.264 codec)
- Resolution: 1920x1080 or 1280x720
- Duration: 30-60 seconds
- Size: Under 10MB recommended
